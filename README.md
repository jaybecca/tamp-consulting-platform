## Business Value

This project demonstrates how businesses can automate the deployment of their websites using modern DevOps practices.

By containerizing the TAMP Consulting website with Docker and implementing a Jenkins CI/CD pipeline, website updates can be deployed faster, more reliably, and with less manual effort.

Key business benefits include:

- Faster website updates and releases
- Reduced deployment errors through automation
- Consistent environments across development and production
- Improved operational efficiency
- Scalable deployment process for future growth
- Better collaboration between development and operations teams

This approach enables organizations to maintain a professional online presence while reducing the time and effort required to manage website deployments.


## Project Overview

TAMP Consulting is a Human Resources and Talent Management firm based in Victoria Island, Lagos, Nigeria.

This project involved designing and deploying a responsive corporate website that showcases the company's services, including:

- Talent Acquisition
- Recruitment Services
- HR Outsourcing
- Training and Development
- Workforce Solutions

The website was containerized using Docker and deployed through an automated Jenkins CI/CD pipeline to demonstrate modern DevOps deployment practices.
